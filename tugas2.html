Proses diskritisasi dengan equal width dan equal frequency#

Diskritisasi#

Diskritisasi digunakan untuk mengubah atribut numerik menjadi atribut kategorikal. Perubahan tersebut dilakukan dengan mengkategorikan atribut numerik menjadi beberapa tingkatan atribut kategorikal. Dalam implementasinya, diskritisasi memiliki dua pendekatan, equal-width intervals dan equal-frequency intervals.

Equal Width Intervals#

Equal-width intervals adalah diskritisasi yang membagi data numerik menjadi beberapa kelompok dengan lebar kelompok yang kurang lebih sama besar. Fungsi cut digunakan untuk mencari interval menggunakan metode Equal Width Intervals

Equal Frequency Intervals#

Equal frequency intervals adalah discretization yang membagi data numerik menjadi beberapa kelompok dengan jumlah anggota yang kurang lebih sama besar. Fungsi qcut digunakan untuk mencari interval menggunakan metode Equal Frequency Intervals

Equal Frequency Intervals lebar kelopak bunga Iris

# Untuk menghitung banyaknya kategori label Sepal Width
jumlahk = len(labelK)

# untuk menghitung interval equal-frequency
dt_lebarsepal_ef = pd.qcut(lebar_sepal, jumlahK, labelK)

Entropi#

Entropi adalah nilai informasi yang menyatakan ukuran ketidakpastian(impurity) dari atribut dari suatu kumpulan obyek data dalam satuan bit.

\[\begin{split} \begin{align*} \displaystyle Entropy(S) &= \sum_{i=0}^{n} -phi \ log_{2} \ phi \\ Keterangan &: \\ S &= Himpunan \ dataset \\ n &= Banyaknya \ partisi \ S \\ phi &= Proporsi \ dari \ S_{i} \ terhadap \ S \end{align*} \end{split}\]
# Data yang akan digunakan
targetdata = df_lebarsepal_ew
# Label Kategori
label = ['Kecil', 'Sedang', 'Lebar']

#fungsi yang digunakan untuk menghitung banyak data berdasarkan kategori
def hitung_kategori(kolom_data, label, kolom, kategori):
  grup = kolom_data.groupby(kategori).count()
  jumlah_kategori = []
  for label in label:
    jumlah_kategori.append(grup.loc[label, kolom])
  return jumlah_kategori
  

hitung_kategori(targetdata, label, targetdata.columns[0], targetdata.columns[1])

#fungsi menghitung nilai entropy
def entropy(targetdata):
    all_prob = []
    for prob in targetdata:
        if (prob/sum(targetdata) != 0):
            all_prob.append(prob/sum(targetdata) * log2(prob/sum(targetdata)))
        else:
            all_prob.append(0)
    return -(sum(all_prob))

#fungsi yang digunakan untuk proses info D
def split(nilai,kolom_data, label, kolom, kategori):
  less_group =kolom_data[kolom_data[kolom] < nilai]
  greater_group =kolom_data[kolom_data[kolom] >= nilai]

  length_less_group = hitung_kategori(less_group, label, kolom, kategori)
  length_greater_group = hitung_kategori(greater_group, label, kolom, kategori)

  return (length_less_group, length_greater_group)

\[ Info_A(D) = \frac{|D_1|}{|D|} Entropy (D_1)\frac{|D_2|}{|D|}+ Entropy (D_2)\]
# fungsi menghitung nilai info
def info(d, targetdata):
    temp = []
    for value in d:
        temp.append((sum(value) / targetdata.shape[0]) * entropy(value))
    return sum(temp)

# fungsi menghitung nilai gain
